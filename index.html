Tauri window position and size testing

  1. The Get button determines the current window position and size values and updates the fields based on these values.
  2. The Set button sets the window's position and size based on the field values. While this is not documented, Tauri handles the position as outer position and size as inner size.
  3. The Get inside event handler button opens a new window based using the field values, determines the new windows position and size values in a tauri://created handler and then updates the fields based on these values.
